# Trasty Pipeline Ingestion Template for sGTM

Template oficial do Google Tag Manager Server-Side para envio de eventos de jornada normalizados à API do Trasty Pipeline.

## Instalação
Este modelo faz parte da Galeria de Modelos da Comunidade do GTM. Para instalar, busque por "Trasty Pipeline Ingestion" na Galeria do seu container do GTM Server-Side.

## Arquivos Estruturados para a Galeria
Esta pasta contém os arquivos necessários para a publicação na Galeria de Modelos de Comunidade do GTM.
- `template.tpl`: O template exportado do GTM contendo o código e permissões.
- `metadata.yaml`: Arquivo de controle de versão.
- `LICENSE`: Licença Apache 2.0 obrigatória para a comunidade.
